FIRST-RUN SETUP

Connect this plugin to your claude.ai Max account. Credentials are stored in the macOS Keychain — never in plaintext on disk, never sent anywhere except claude.ai.

  1. 01

    Open claude.ai in your browser

    Sign in normally with your Anthropic account. Once you see the chat UI, keep that tab open.

    ↗ open claude.ai
  2. 02

    Copy your sessionKey cookie

    Open browser DevTools (Cmd+Option+I in Chrome/Safari) → Application tab → Cookies → https://claude.ai. Find the row named sessionKey and copy its full value. 

    sessionKey  sk-ant-sid01-…  HttpOnly  Secure

    ⚠ This cookie is equivalent to a password. Anyone with it can act as you on claude.ai. Treat it accordingly.

  3. 03

    Find your org id

    In the same DevTools panel, go to Network, click any request to /api/organizations/<uuid>/…, and copy the UUID. (Or just open any chat and look at the URL.) 

    orgId     00000000-0000-0000-0000-000000000000
  4. 04

    Paste and verify

    advanced // fingerprint headers (optional)

    Some claude.ai endpoints reject requests missing these. Pull them from any /api/organizations/… request in DevTools → Network → Headers if the basic setup doesn't work.
